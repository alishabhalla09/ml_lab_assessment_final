# ✨ Project Features

This project provides several powerful tools for Human Activity Recognition:

### 1. Readable Feature Names
The AI uses 561 different sensors. Usually, these have confusing names like `tBodyAcc-mean()-X`. Our system automatically changes them to easy names like `Body Acceleration Mean X`.

### 2. Interactive Dashboard
A beautiful, easy-to-use interface built with Streamlit. It includes:
- **Cluster Charts**: See how the AI groups different activities.
- **Heatmaps**: Compare what the AI thinks vs. what people actually did.
- **Radar Charts**: See the "fingerprint" of each activity.

### 3. Activity Prediction Engine
Type in sensor values and the AI will immediately tell you if the person was likely Walking, Standing, or Laying down.

### 4. Smart Data Processing
- Handles missing data automatically.
- Scales data for better AI performance.
- Uses PCA to make complex data easy to visualize.
