# 🛠️ Installation & Setup

To get HAR Studio Pro running on your local machine, follow these steps:

### 1. Requirements
Ensure you have Python 3.8+ installed.

### 2. Install Packages
Run the following command in your terminal to install all required libraries:
```bash
pip install streamlit pandas numpy matplotlib seaborn scikit-learn
```

### 3. Data Preparation
Make sure your `data/` folder contains `train.csv` and `test.csv`.

### 4. Running the Project
The project has two main parts:
- **Analysis**: Processes data and creates the AI models.
- **Dashboard**: Shows the results and lets you predict activities.
